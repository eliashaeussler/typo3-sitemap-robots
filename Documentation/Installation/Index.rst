..  _requirements:

Requirements
============

-   PHP 8.2 - 8.4
-   TYPO3 13.4 LTS

..  _steps:

Installation
============

Require the extension via Composer (recommended):

..  code-block:: bash

    composer require eliashaeussler/typo3-sitemap-robots

Or download it from the
`TYPO3 extension repository <https://extensions.typo3.org/extension/sitemap_robots>`__.
